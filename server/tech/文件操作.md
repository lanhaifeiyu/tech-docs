##### 查看文件列表

```shell
#按修改的时间倒序显示文件的详细信息
ls -lrth
```



#### 移动文件

1. 移动文件夹下所有文件

   ```
   mv [选项(option)] 源文件或目录 目标文件或目录
   ```

   `mv /usr/local/lh/pkg/pkg/* /usr/local/lh/pkg/`

#### 删除文件夹

1. 删除

   ```
   rm [options] name
   ```

- -i 删除前逐一询问确认。
- -f 即使原档案属性设为唯读，亦直接删除，无需逐一确认。
- -r 将目录及以下之档案亦逐一删除。

​       `rm -rf /usr/local/lh/pkg/pkg`

只查看目录信息

ls -ld /tmp
